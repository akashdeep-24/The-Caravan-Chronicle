# 🚀 Caravan Chronicle

A smart civic complaint management system connecting citizens with municipal authorities for efficient issue tracking and resolution.
citizens face delays and lack transparency when reporting civic issues. This project provides a centralized digital platform to solve this.

### Basic Features
- User Registration & Login
- Complaint Submission
- Ticket Lifecycle Management
- Municipal Dashboard

### Advanced Features
- Heatmap Visualization
- SLA Tracking
- Escalation System
- Notifications

## 🛠️ Tech Stack
Frontend: React.js  
Backend: Node.js, Express  
Database: MongoDB  
Tools: Git, GitHub
